1. **Clone this repo:**

```bash
git clone https://github.com/AdarshKumarSr/BurrowAI.git

```

2. **Setup frontend:**

```bash

```

3. **Setup backend:**

```bash

npm install
node server.js
```

```bash
cd ../backend

python app.py
```
